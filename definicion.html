Definición

Una aplicación web progresiva o PWA es un software de aplicación que se basa en tecnologías como HTML, CSS y JavaScript. Es decir, una PWA es una página web que se ve y se comporta como una aplicación móvil normal.


¿Cómo funciona PWA?

Tal y como se ha dicho, un PWA es una página web que funciona y se parece a una aplicación móvil. Usar un PWA es más fácil que una aplicación móvil porque los usuarios no necesitan instalar nada.


¿Cuál es la diferencia entre las aplicaciones normales, PWA y las páginas web tradicionales?

La diferencia principal entre las aplicaciones normales, las aplicaciones web progresivas (PWA) y los sitios web tradicionales está en su capacidad. En comparación con el sitio web tradicional, las aplicaciones móviles normales se inician rápidamente y tienden a cargarse, incluso con poca o ninguna disponibilidad de red. Estas aplicaciones y PWA pueden enviar notificaciones y compartir la ubicación, ventajas que hasta ahora la web no había ofrecido a los usuarios. PWA tiene la misma capacidad que una aplicación móvil y mejora el uso de la web, pero no necesita utilizar espacio de almacenamiento en el móvil por descargar una aplicación.
